## Project Overview
This dashboard provides a 360-degree view of business operations, allowing stakeholders to monitor key performance indicators (KPIs) and drill down into specific segments, categories, and regions.

### Key Insights & Features
* **Performance Tracking:** Real-time visibility into Total Sales, Profit Margins, and Quantity Sold.
* **Geographic Mapping:** Visualizing regional profit distribution to identify high-performing markets vs. underperforming areas.
* **Product Analysis:** Comparative breakdown of Furniture, Office Supplies, and Technology categories.
* **Customer Segmentation:** Analysis of sales across Corporate, Home Office, and Consumer segments.
* **Interactive Filtering:** Global filters for Year, Region, and Segment to enable customized data exploration.

## Technical Implementation
* **Tool:** Tableau Desktop
* **Data Source:** Superstore Sales Dataset
* **Functions Used:** LOD Expressions, Calculated Fields, Action Filters, and Dual-Axis Charts.
* **Publishing:** Managed via Tableau Public for cloud-based accessibility.
